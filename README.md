# 단위 변환기 (Unit Converter)

다양한 단위를 쉽고 빠르게 변환할 수 있는 심플한 웹 애플리케이션입니다.

## 주요 기능

- **길이 변환**: cm, inch, feet, mile, km, m 등
- **무게 변환**: kg, pound 등
- **부피 변환**: 리터, 갤런 등

각 변환 항목별로 입력 폼이 제공되며, 값을 입력하고 버튼을 클릭하면 변환 결과가 즉시 표시됩니다.

## 기술 스택

- HTML / CSS / JavaScript (순수 웹 기술, 별도 프레임워크 없음)
- 데이터베이스 없음

## 설치 및 실행 방법

별도의 설치 과정 없이 바로 실행할 수 있습니다.

### 1. 저장소 클론

```bash
git clone <저장소 URL>
cd test-web-service
```

### 2. 실행

`index.html` 파일을 브라우저에서 직접 열거나, 간단한 로컬 서버를 통해 실행합니다.

**브라우저에서 직접 열기:**
```
index.html 파일을 더블클릭하거나 브라우저로 드래그
```

**로컬 서버로 실행 (Python):**
```bash
# Python 3
python3 -m http.server 8080

# 브라우저에서 접속
# http://localhost:8080
```

**로컬 서버로 실행 (Node.js):**
```bash
npx serve .

# 브라우저에서 접속
# http://localhost:3000
```

## 사용 방법

1. 변환하고 싶은 단위 항목(길이, 무게, 부피)의 폼을 선택합니다.
2. 변환할 값을 입력합니다.
3. 변환할 단위를 선택합니다.
4. **변환** 버튼을 클릭하면 결과가 옆에 표시됩니다.
