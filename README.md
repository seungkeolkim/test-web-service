# 단위 변환기 (Unit Converter)

다양한 단위를 쉽고 빠르게 변환할 수 있는 심플한 웹 애플리케이션입니다.  
별도의 회원가입, 로그인, 데이터베이스 없이 브라우저만 있으면 즉시 사용할 수 있습니다.

---

## 프로젝트 개요

일상에서 자주 마주치는 단위 변환 작업을 간편하게 처리할 수 있는 도구입니다.  
길이·무게·부피·온도·속도·면적·데이터 크기 일곱 가지 카테고리에 대해 각각 입력 폼을 제공하며,  
값을 입력하고 변환 버튼을 누르면 변환 결과가 즉시 화면에 표시됩니다.

| 카테고리 | 지원 단위 |
|----------|-----------|
| 길이 | cm, m, km, inch, feet, mile |
| 무게 | kg, pound (lb) |
| 부피 | 리터 (L), 갤런 (gal) |
| 온도 | °C, °F, K |
| 속도 (Speed) | km/h, mph, m/s, knot |
| 면적 (Area) | m², ft², 평(pyeong) |
| 데이터 크기 | B, KB, MB, GB, TB |

---

## 주요 기능

### 길이 변환
- **cm (센티미터)** ↔ inch, feet, m, km, mile
- **m (미터)** ↔ cm, km, inch, feet, mile
- **km (킬로미터)** ↔ m, cm, mile
- **inch (인치)** ↔ cm, feet, m
- **feet (피트)** ↔ inch, cm, m
- **mile (마일)** ↔ km, m

### 무게 변환
- **kg (킬로그램)** ↔ pound (lb)
- **pound (파운드)** ↔ kg

### 부피 변환
- **리터 (L)** ↔ 갤런 (gal)
- **갤런 (gal)** ↔ 리터 (L)

### 온도 변환
- **°C (섭씨)** ↔ °F (화씨), K (켈빈)
- **°F (화씨)** ↔ °C (섭씨), K (켈빈)
- **K (켈빈)** ↔ °C (섭씨), °F (화씨)
- 온도는 선형 비례가 아닌 공식 기반 변환을 사용합니다.
  - °C → °F : `F = C × 9/5 + 32`
  - °C → K  : `K = C + 273.15`
  - °F ↔ K  : 섭씨를 중간 단위로 경유하여 변환

### 속도 변환
- **km/h (킬로미터/시)** ↔ mph, m/s, knot
- **mph (마일/시)** ↔ km/h, m/s, knot
- **m/s (미터/초)** ↔ km/h, mph, knot
- **knot (노트)** ↔ km/h, mph, m/s
- 모든 단위는 m/s(미터/초)를 기준 단위로 상호 변환합니다.

### 면적 변환
- **m² (제곱미터)** ↔ ft² (제곱피트), 평
- **ft² (제곱피트)** ↔ m², 평
- **평 (평방)** ↔ m², ft²
- 1평 = 3.305785 m², 1 ft² = 0.09290304 m²
- 부동산·인테리어 분야에서 자주 사용하는 한국식 면적 단위(평)를 지원합니다.

### 데이터 크기 변환
- **B (바이트)** ↔ KB, MB, GB, TB
- **KB (킬로바이트)** ↔ B, MB, GB, TB
- **MB (메가바이트)** ↔ B, KB, GB, TB
- **GB (기가바이트)** ↔ B, KB, MB, TB
- **TB (테라바이트)** ↔ B, KB, MB, GB
- 1 KB = 1,024 B (이진 기반, IEC 표준)로 계산합니다.

## 인터랙션 효과 (특수 기능)

일반적인 변환 기능 외에, 페이지 사용 경험을 풍부하게 만드는 시각적 효과들이 포함되어 있습니다.

### 랜덤 배경 이미지
- 페이지를 로드할 때마다 [picsum.photos](https://picsum.photos)에서 무작위 사진을 배경으로 표시합니다.
- 새로고침할 때마다 다른 풍경·사진이 나타납니다.
- 이미지 로딩 실패 시에는 `style.css`에 정의된 폴백 배경색(`#f0f2f5`)이 유지됩니다.

### 클릭 폭발 이펙트
- 화면 어디를 클릭해도 해당 위치에서 폭발 애니메이션이 재생됩니다.
- 중심 플래시(flash) 1개와 방사형으로 퍼지는 파편(particle) 8개로 구성됩니다.
- 애니메이션 완료 후 DOM에서 자동 제거되어 메모리 누수를 방지합니다.

### 우클릭 컨텍스트 메뉴 비활성화
- 마우스 우클릭 시 브라우저 기본 컨텍스트 메뉴가 표시되지 않습니다.

## 브라우저 호환성

이 프로젝트는 다음 CSS 기능을 사용합니다:

| 기능 | 용도 |
|------|------|
| `backdrop-filter: blur()` | 헤더 및 카드의 반투명 블러 효과 |
| CSS 커스텀 프로퍼티 (`--particle-angle`) | 폭발 파티클 방향 제어 |
| CSS `@keyframes` 애니메이션 | 클릭 폭발 이펙트 |

위 기능들은 **최신 브라우저(Chrome, Edge, Safari, Firefox 최신 버전)** 를 권장합니다.  
Internet Explorer는 지원하지 않습니다.

> `backdrop-filter`는 Firefox의 경우 설정에서 활성화해야 할 수 있습니다.

## 설치 및 실행 방법

별도의 설치 과정 없이 바로 실행할 수 있습니다.

### 1. 저장소 클론

```bash
git clone <저장소 URL>
cd test-web-service
```

### 2. 실행

#### ✅ 권장 방법: run.sh 스크립트 사용

Python3가 설치된 환경에서 가장 간단하게 실행할 수 있습니다.  
서버 시작 후 브라우저가 자동으로 열립니다.

```bash
# 기본 실행 (포트: 8080)
bash run.sh

# 포트 변경 시 PORT 환경변수 지정
PORT=3000 bash run.sh
```

> 브라우저에서 `http://localhost:8080` (또는 지정한 포트)으로 자동 접속됩니다.  
> 종료하려면 `Ctrl+C`를 누르세요.

---

#### 그 외 실행 방법

**브라우저에서 직접 열기:**
```
index.html 파일을 더블클릭하거나 브라우저로 드래그
```

**로컬 서버로 실행 (Python):**
```bash
# Python 3
python3 -m http.server 8080

# 브라우저에서 접속
# http://localhost:8080
```

**로컬 서버로 실행 (Node.js):**
```bash
npx serve .

# 브라우저에서 접속
# http://localhost:3000
```

**VS Code Live Server 확장 사용:**
```
1. VS Code에서 프로젝트 폴더를 열기
2. Live Server 확장 설치 (Extensions > "Live Server" 검색)
3. index.html 파일에서 우클릭 → "Open with Live Server" 클릭
4. 브라우저에서 http://127.0.0.1:5500 자동 접속
```

## 사용 방법

1. 변환하고 싶은 카테고리(길이 / 무게 / 부피 / 온도 / 속도 / 면적 / 데이터 크기)의 폼을 찾습니다.
2. 변환할 값을 입력 필드에 입력합니다.
3. 변환할 원본 단위와 목표 단위를 선택합니다.
4. **변환** 버튼을 클릭하거나 **Enter 키**를 누르면 결과가 폼 옆에 즉시 표시됩니다.

> **키보드 단축키:** 숫자 입력 후 Enter 키를 누르면 버튼 클릭 없이 바로 변환이 실행됩니다.
